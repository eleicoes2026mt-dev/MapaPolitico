# CampanhaMT - Gestão Eleitoral MT

Sistema de gestão eleitoral para campanha no Mato Grosso: dashboard, assessores, apoiadores, votantes, benfeitorias, mensagens, dados TSE, estratégia e mapa regional.

**Stack:** Flutter Web, Supabase (Auth, DB, Storage), Riverpod, GoRouter.

- **Subir no GitHub e deploy:** [docs/GITHUB_E_VERCEL.md](docs/GITHUB_E_VERCEL.md)
- **Deploy automático (GitHub Action → Vercel):** [docs/GITHUB_ACTION_VERCEL.md](docs/GITHUB_ACTION_VERCEL.md) — use isso porque a Vercel não tem Flutter no servidor.
- Deploy Vercel (detalhes): [docs/DEPLOY_VERCEL.md](docs/DEPLOY_VERCEL.md)
- Supabase CLI: [docs/SUPABASE_CLI.md](docs/SUPABASE_CLI.md)
